Problem Statement : Nowadays, almost every big platform requires the users to identify themselves using a loginID and password. In such cases, passwords play a major role in securing our identity. Therefore, the password that we create should be difficult to guess, in order to provide maximum security to our digital identities.
Project Scope : This file can be used to generate random, difficult to guess passwords which can provide stronger security to our digital identities.
Target Users : Any person using popular digital platforms like GMail, whatsapp, Instagram, LinkedIn ;etc. can use this tool to generate strong passwords for their digital accounts.
Feature : It provides the users with strong passwords using base words entered by the users themselves.
