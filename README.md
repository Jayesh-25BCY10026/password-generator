PASSWORD GENERATOR
Project Overview : A password generator created using python which takes user input as base words to create a safe password.
Feature : The user can enter any base words of their choice to get a strong password that can be used on any platform.
Tools Used : Python, 'random' module (a built-in python module)
Instructions : 1.> Open the file using Python.
               2.> Enter any base words of your choice as input.
               3.> Get the generated password and use it on the desired platform.
